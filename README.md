試著撈所有區域：
思考點：資料內區域是完整地址，不是區域，像是隨便一筆資料的地址是「高雄市806前鎮區漁港中一路2號地下一樓」，但我只需要撈出「前鎮區」三個字而已。
大致解決辦法應該是：
1. 跑回圈抓區域
2. 將不同的區域塞進某個陣列內

區域抓法：
1. 抓「數字」後的三個字
2. 看到區後，把前面兩個字也抓進來 

上面都只適合區的字數沒有超過3ＸＤ，嚴謹一點的抓髮應該是看到「數字」後，抓後面的字直到「區」這個字。 <br>
感覺也可以把數字前的字串弄消失，區後面的字串消失
